# Company Fit：我司适配分析

> 把 wiki/topics 与 analysis/trends 翻译成对我司的影响。
> 输入：`ops/company-context.md`（脱敏的我司背景）。
> 输出：报告 §8 的素材。

## 文件清单

| 文件 | 内容 |
|---|---|
| `capability-map.md` | 我司当前在数据编织能力域的位置（自评） |
| `opportunities.md` | 6G 数据面相关的机会窗（短/中/长期） |
| `threats.md` | 主要风险与挑战 |
| `actions.md` | 建议行动清单（带时间窗与负责人占位） |
| `talents.md` | 人才/团队能力差距 |
| `partnerships.md` | 建议关注的合作 / 收购 / 标准参与 |

## 写作约束

- 不要做"诺贝尔奖式建议"。建议必须是**我司可执行**的尺度
- 区分**短期（6 个月）/ 中期（12–18 月）/ 长期（24+ 月）**
- 每条建议给出：**为什么、做什么、不做什么、衡量指标**
- 引用 wiki/topics 和 analysis/trends，每条建议至少 1 个证据链
