# Data Mesh

> 一种以领域为中心的分布式数据架构范式，强调数据所有权去中心化和联邦式治理。

**中文**：数据网格
**英文**：Data Mesh
**缩写**：无

## 定义

Data Mesh（数据网格）是由 Zhamak Dehghani 于 2019 年提出的数据架构范式，基于四大原则：领域数据所有权（Domain Ownership）、数据即产品（Data as a Product）、自助数据基础设施平台（Self-serve Data Infrastructure Platform）和联邦计算治理（Federated Computational Governance）。与数据编织的技术驱动路径不同，数据网格更强调组织结构和文化变革。
