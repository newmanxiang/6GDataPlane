## 1. 公司画像

- **公司**：中兴通讯股份有限公司（ZTE），A+H 股上市（000063.SZ / 0763.HK）
- **行业定位**：全球综合性通信信息解决方案提供商，涵盖运营商网络、政企业务、消费者业务三大板块；近年强化"数字经济全能力"与算力基础设施
- **规模量级**：2025 年营收 1339.0 亿元（同比 +10.4%），归母净利润 56.2 亿元；其中**算力相关营收同比 +150%**（增长最快板块）；全球员工数万级
- **主要客户类型**：国内三大运营商（尤以中国移动、中国电信合作密切）、海外运营商、政企行业客户、OEM/ODM 伙伴
- **核心市场**：国内（主力）+ 海外（百级以上国家和地区），行业垂直（能源、交通、政务、金融等）

## 2. 当前在数据领域的能力

- **已有数据平台 / 产品**：面向运营商的网络大数据平台、数据中台；**SDI（Service & Data Intelligence，服务与数据智能）**产品线是数据能力的组织载体
- **已建数据治理能力**：网络运维数据治理（告警/性能/配置），元数据管理有实践；**uSmartNet 自主进化网络**（网元内生 / 单域自治 / 跨域协同）承载跨域数据编排
- **已有 AI/ML 平台**：AI 推理/训练服务器产品线；**星云智算网络**（AI 算力承载）；网络智能化 AI（智能运维、网优）；算力营收 2025 同比 +150%
- **数据虚拟化 / 数据编排能力**：网络数据层面有跨域编排（uSmartNet），完整数据虚拟化层仍在演进
- **数据隐私 / 联邦能力**：跟随国内数据安全法规，运营商场景有隐私保护实践；联邦学习结合 NWDAF 有探索
- **NWDAF / MDAS / KPI 服务**：5G Common Core 已含 NWDAF 实现（"全栈 AI+ 核心网"四大智能，2025），网络 KPI 分析服务面向运营商交付
- **自智网络（AN）**：**AIR Net** 高阶自智网络方案，核心是**数智引擎**——集成 **数据引擎 + AI 大模型引擎（智能引擎）+ 数字孪生引擎**三大组件（《自智网络白皮书 2025》）。Fault Agent（跨域/单域）以知识图谱+大模型做多域诊断；对齐 TM Forum AN L4+ 与 KCI/KEI/KBI + BVM；与中国电信多省联合验证
- **数据中台的架构落点**：数据中台 = 自智网络三引擎中的**"数据引擎"**——这是数据中台在中兴网络体系中的既定位置，也是其向"6G 数据面"延伸的天然支点（数据引擎 → 6G 数据服务化组件）

## 3. 当前在 6G/通信领域的能力

- **5G/5G-A**：全球主要 5G 设备商之一，5G-A 商用方案已发布
- **标准组织参与**：深度参与 3GPP（含 SA2 6G 数据框架 WT#5 提案）、ETSI（含 **ZSM 029 数据管理代理**）、ITU（与中国移动主导 **ITU-T SG15 首个 800G MTN——首个 6G 传输国际标准**）、IMT-2030(6G) 推进组、O-RAN
- **空口 / 核心网 / 边缘云 产品**：全线；RAN、核心网（Common Core）、边缘云（MEC）；SPN/承载网领先
- **6G 旗舰项目**：发布 6G 白皮书系列（含 AI Core Technology System）；6G 战略"两大变革方向（AI×通信深度融合 Native AI、空天地一体）+ 四大演进方向"；旗舰原型 **GigaMIMO**（集中式 2048 阵元 U6G 频段 6425–7125MHz，小区容量 10×；分布式与中国移动商网验证边缘体验 6×；频谱效率 +30%+）
- **AI-RAN / 智算网络**：**AIR / AIR MAX** 端到端方案 + **AIREngine 智算板**（2024，站点本地 AI 推理）；NodeEngine（2020）→ UniEngine → AIREngine 演进链；提出 "AI for RAN / RAN for AI" 双维框架

## 4. 战略意图（已确认）

> 视角：**数据中台团队 / 战略专家组**——从数据中台切入，结合 6G 数据面机会，深度嵌入 6G 业务以获取更大发展空间。

- **核心目标**：以现有**数据中台**为支点，抓住 6G 数据面机会窗，把数据中台能力"嵌入"6G 业务链条（而非孤立做企业级数据编织中台），从而放大数据团队在公司内的战略地位与发展机会
- 短期（6 个月）期望：把数据中台/SDI 既有资产（KG、数据治理）与 6G 数据面叙事挂钩，形成对内说服力
- 中期（12-18 月）期望：在 3GPP SA2 WT#5、ETSI ZSM 029 等入口贡献提案，把数据中台能力标准化为"6G 数据面"组件
- 长期（24+ 月）战略定位：数据中台作为自智网络"数据引擎"的支撑能力，在 6G AI 原生/数据服务化场景中获得更深嵌入机会
- 已识别的关键威胁：地缘政治与供应链/实体清单风险、海外市场准入限制、AI 算力生态竞争、华为在数据面（DaaS）标准话语权领先、**数据中台在设备商内部易被边缘化**
- 已识别的关键机会：6G 标准窗口先发；数据编织×6G 的电信适配空白（尤其 RAN/终端侧）；数据中台从"支撑角色"升级为"6G 业务核心组件"的窗口

## 5. 资源边界（已确认，直接约束"可执行尺度"）

- **可投入资金量级**：**年投入千万级别人民币（~10-50M RMB/年）**——属"一个团队 + 若干 PoC"尺度，**不足以支撑独立产品线、大规模 GPU 集群或并购**
